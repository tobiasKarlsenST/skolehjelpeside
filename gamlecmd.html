CMD/PowerShell Commands

Kjør CMD som administrator!

Commands som skanner og fikser korrupte filer
Command Beskrivelse
sfc /scannow
 Skanner og fikser alle beskyttede Windows systemfiler for korrupsjon.
Dism /Online /Cleanup-Image /CheckHealth
 Sjekker raskt om Windows-image har korrupsjon (kun rapport).
Dism /Online /Cleanup-Image /ScanHealth
 Dypere skann enn CheckHealth.
Dism /Online /Cleanup-Image /RestoreHealth
 Skanner og fikser korrupte filer via Windows Update eller lokal kilde.
Dism kan fikse ting som SFC ikke kan, men det er ofte greit å teste SFC først.

Info Commands
Command Beskrivelse
systeminfo
 Teknisk info om maskinens software, hardware og OS.

 Viser Windows-versjon.
